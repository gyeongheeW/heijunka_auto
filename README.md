# Heijunka Production Planning System

## 📊 프로젝트 개요
**Heijunka 생산 계획 관리 시스템** - 제조업 생산 라인의 작업 순서 최적화 도구

이 프로젝트는 전통적인 제조업 생산 계획에 특화된 도구로, 실제 현장에서 사용될 수 있는 실용적인 기능을 제공합니다.

## 🏗️ 기술 스택
- **언어**: C# .NET 8.0 (WPF 데스크톱 애플리케이션)
- **최적화 엔진**: Google OR-Tools (선형 프로그래밍)
- **UI 프레임워크**: WPF (Windows Presentation Foundation)
- **데이터 처리**: ClosedXML (Excel 파일 처리)
- **빌드**: Single-file self-contained 배포

## 🔧 주요 기능

### 1. 타임펜스(Time Fence) 계획
- 생산 일정을 타임 단위로 분할하여 관리
- 각 타임펜스별 생산 목표량 설정
- 수요/공급 균형 최적화
- Google OR-Tools SCIP 솔버 사용 (혼합 정수 계획법)

### 2. 시퀀싱(Sequencing)
- 생산 품목의 작업 순서 최적화
- 전환 시설(transfer) 비용 최소화
- 제품 분류(class) 혼합 균형화
- 지원 알고리즘: Greedy, Swap, Tabu Search

### 3. LP(선형 프로그래밍) 기반 최적화
- 제약 조건: 재고, 수요, 생산 능력, QPC(수량당판)
- 목적 함수: 수요 충족, 재고 최소화, 비용 최소화
- QPC(Quantity Per Container) 단위 생산량 제약 지원

## 🏗️ 아키텍처 구조

### 데이터 모델
- `Item`: 품목 정보 (코드, 분류, 안전재고)
- `Order`: 생산 오더 (품목코드, 수량, 분류)
- `PlanSettings`: 계획 설정 (UPH, 시프트, 분류 가중치)
- `AppConfig`: 애플리케이션 설정 (파일 경로, 알고리즘 선택)


### 분류 및 가중치 시스템
- 7단계 분류: 분류A~분류G
- 가중치 설정: { 2, 4, 1, 1, 1, 1, 1 }
- 동일 분류 패널티: -99점
- 전환 규칙 가중치: 10점
- 분류 혼합 가중치: 1점

## ⚙️ 설정 기능

### 생산 설정
- UPH(Units Per Hour): 시간당 생산량 (기본값: 40)
- QPC(Quantity Per Container): 컨테이너당 수량
- 시프트 분할: 3교대, 시프트당 4펜스
- 시기별 시작시간: 08:00, 16:00, 00:00
- 펜스 당 시간: 2시간 (기본값)

### 알고리즘 옵션
- **Greedy**: 그리디 알고리즘 (빠른 실행)
- **Swap**: 국소 최적화 스왑 알고리즘
- **Tabu**: 타부 서치 (탐색 최적화)

### 제약 조건
- 최대 반복 횟수: 1000회 (기본값)
- 개선 없음 한계: 50회 (기본값)
- 최대 주문 수량: 20개 (선택적 적용)

## 📤 입력/출력 시스템

### 입력 파일
- Master File: 품목 마스터 데이터 (코드, 분류, 안전재고)
- Transfer File: 제품 간 전환 비용 테이블
- Demand File: 시기별 수요 예측 데이터

### 출력 파일
- 타임펜스 계획 결과
- 시퀀싱 결과 (최적 작업 순서)
- 품질 점수 리포트 (전환 비용, 분류 혼합 점수)

### 데이터 교환
- Excel 파일 입출력 (.xlsx 형식)
- 클립보드 복사/붙여넣기 지원
- JSON 설정 파일 자동 저장/로드

## 🎯 실제 적용 시나리오

### 제조업 생산 계획 워크플로우
1. **데이터 준비**: Excel로 품목 마스터, 수요, 전환 테이블 준비
2. **계획 설정**: UPH, 시프트, 분류 가중치 설정 구성
3. **타임펜스 계획**: OR-Tools로 최적 생산량 분배 계산
4. **시퀀싱**: 선택된 알고리즘으로 작업 순서 결정
5. **품질 평가**: 전환 비용, 분류 혼합 품질 평가
6. **결과 출력**: Excel 파일로 최종 작업 순서 생성 및 리포트

### 주요 비즈니스 가치
- **전환 시간 최소화**: 제품 간 생산 전환 시 발생하는 설비 세팅 시간 최소화
- **분류 혼합 최적화**: 균일한 품질 유지 및 생산 안정성 향상
- **재고 최소화**: 최소 재고 유지 및 수요 충족 균형 유지
- **라인 이용률 극대화**: 생산 라인의 가동률 향상
